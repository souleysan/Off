### Types de contributions
>- Corrections (fautes, erreurs, modifications, suppressions, ajouts)
>- Re-Lectures (tickets, Merge Requestion, MAJ)
>- Analyses et critiques (toujours constructives, c'est le bon mood ici)

<br>

### Contribution au projet - 8 Americain
Avant et après chaque modifications, il faudra s'assurer que vos modifications n'apportent aucun conflits (sous peine de repréndre vos fichiers et de ne pas dormir de la nuit)
<br>

#### Les étapes à suivre

Afin de contribuer au projet, il va falloir **cloner le dépôt en local (sur votre machine)**

```
git clone https://github.com/souleysan/Off.git
``` 
<br>

et **descendez sur la branche** qui définit la fonctionnalité à laquelle vous souhaitez contribuer

```
git checkout **nom-de-la-branche-fonctionnalité**
```
<br>

Une fois vos modifications apportées, **ajoutez votre/vos fichier.s** à l'index

```
git add **nom-du-fichier-a-ajouter**
```
<br>

Créez un **commit** qui identifie facilement vos modifs

```
git commit -m "Message de commit"
```
<br>

Une fois tout validé (**SANS CONFLITS**), vous **poussez** sur la branche actuelle (celle sur laquelle vous êtes descendu avec **git checkout**):

```
git push origin **nom-de-la-branche-actuell**
```
